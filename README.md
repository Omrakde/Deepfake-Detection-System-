# Deepfake-Detection-System-
-Implemented image preprocessing, feature extraction, model evaluation, and REST API deployment for real-time deepfake detection.
-Utilized Python, TensorFlow, OpenCV, NumPy, and Pandas for model development, data preprocessing, and inference
– Developed a deep learning-based system to detect deepfake videos using facial, audio, and emotion-based feature analysis.
– Processed and preprocessed 1,000+ video samples using OpenCV and feature extraction techniques to classify content as real or fake.
– Built and evaluated machine learning/deep learning models for binary classification, achieving high detection accuracy.
– Deployed the backend using Flask/FastAPI, enabling real-time prediction through REST APIs.
